# Checklist Kesiapan Penelitian

## 1. Administrasi Penelitian

- [ ] Proposal penelitian final.
- [ ] Surat tugas/tim penelitian.
- [ ] Ethical clearance.
- [ ] Izin lokasi penelitian.
- [ ] Izin penggunaan instrumen.
- [ ] Informed consent.
- [ ] SOP pengambilan data.
- [ ] SOP perlindungan data pasien.

## 2. Instrumen

- [ ] Kuesioner De Jong Gierveld Loneliness Scale 11 item sudah sesuai.
- [ ] Terjemahan/proofread bahasa Indonesia sudah disetujui tim peneliti.
- [ ] Skoring emotional loneliness sudah diverifikasi.
- [ ] Skoring social loneliness sudah diverifikasi.
- [ ] Kategori total skor sudah disetujui.
- [ ] Interpretasi hasil sudah divalidasi.
- [ ] Rekomendasi edukasi sudah divalidasi.

## 3. Aplikasi

- [ ] Landing page final.
- [ ] Kalkulator public final.
- [ ] Login admin/petugas final.
- [ ] Data pasien final.
- [ ] Assessment final.
- [ ] Hasil assessment final.
- [ ] Cetak laporan final.
- [ ] Edukasi perawat final.
- [ ] Edukasi keluarga final.
- [ ] Booklet digital final.
- [ ] Admin bisa mengelola booklet.
- [ ] Admin bisa mengelola konten edukasi.
- [ ] Backup database tersedia.

## 4. Uji Coba

- [ ] UAT admin.
- [ ] UAT petugas/perawat.
- [ ] UAT kalkulator public.
- [ ] UAT mobile.
- [ ] UAT cetak laporan.
- [ ] Bug prioritas tinggi sudah selesai.
- [ ] Bug prioritas sedang sudah disepakati.

## 5. Data dan Privasi

- [ ] Data pasien yang digunakan sesuai izin.
- [ ] Identitas pasien dibatasi sesuai kebutuhan penelitian.
- [ ] Akses admin dibatasi.
- [ ] Password tidak dibagikan sembarangan.
- [ ] Backup disimpan di tempat aman.
- [ ] Data tidak ditampilkan di halaman public.

## 6. Pelatihan Pengguna

- [ ] Admin sudah dilatih.
- [ ] Petugas/perawat sudah dilatih.
- [ ] Panduan penggunaan tersedia.
- [ ] Alur assessment sudah dipahami.
- [ ] Cara membaca hasil sudah dipahami.
- [ ] Cara cetak laporan sudah dipahami.

## 7. Serah Terima

- [ ] Source code diserahkan/tersimpan.
- [ ] Akses server diserahkan.
- [ ] Akses database diserahkan.
- [ ] Akun admin diserahkan.
- [ ] Dokumen teknis diserahkan.
- [ ] Manual book diserahkan.
- [ ] BAST ditandatangani.

## 8. Catatan Penting

Sebelum penelitian real dilakukan, tim peneliti perlu memastikan bahwa penggunaan aplikasi tidak melanggar aturan etik, privasi pasien, dan kebijakan institusi/rumah sakit.
